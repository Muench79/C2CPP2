# Camp2Code - Projekt Phase 2

## Inhalt:
- Agende_Projektphase2.pdf (Slides für einführenden Überblick)
- Anleitung_Projektphase2.pdf (Anleitung für Inbetriebnahme des Autos, Installation von Software, Einrichten einer Arbeitsumgebung)
- Anforderungen_Projektphase2.pdf (Beschreibung der Projektaufgabe)
- Ordner: Software 
    - basisklassen.py (Die Basisklassen erlauben die Ansteuerung einzelnen Fahrzeugkomponenten.)
    - basisklassen_cam.py (Basisklasse für Kamera.)
    - config.json (Konfigurationsfile)
    - SimpleRemoteControl.py (einfache Kontrolle mittels Tastatur)
    - DashRemoteControl.py (Kontrolle mittels Dash-App)
    - DashMobilRemoteControl.py (Kontroll mittels Dash-App für mobile Endgeräte mit Touchscreen)
- Ordner: Trainermaterial (Zusatzmaterial für Wiederholungen/Erklärungen etc.)
